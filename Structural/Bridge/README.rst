`Bridge`_
=========

Purpose
-------

Decouple an abstraction from its implementation so that the two can vary
independently. (http://en.wikipedia.org/wiki/Bridge\_pattern)

Sample:
^^^^^^^

-  `Symfony
   DoctrineBridge <https://github.com/symfony/DoctrineBridge>`__
